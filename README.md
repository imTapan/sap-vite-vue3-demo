# sap-vite-vue3-demo
Sap mta Setup using Vite + Vue 3

## How to run?
- Install Dependencies
``` npm i ```

- Make Vite Build
``` npm run build ```

- Copy extension configs form build-archive to .dist folder

- Make mta build
``` npm run mta ```

that's it now you can find mta build at mta_archives directory.
